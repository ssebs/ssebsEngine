This is a test of ssebsEngine.
The .jar file was exported in eclipse. 
======================================

To "Install" this, 

Save this folder somewhere safe.
Then, Create a link/shortcut to the Desktop on the _Launch file. 
	If on Windows, use _Launch.bat, otherwise, use _Launch.sh
Double Click on the _Launch file.
	If that doesn't work, open a terminal to the file and run ./_Launch.<sh/bat>
	If you get a permission denied error, set the permissions: 
		sudo chmod 755 _Launch.<sh/bat>
